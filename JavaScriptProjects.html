API Server for Northcoders


API Server:

https://my-news-api.herokuapp.com/api

GitHub:

https://github.com/karben86/nc_news


Description

The aim of this project was to design an API server for the purpose of accessing news data programmatically similarly to a service such as reddit. This information should then be available to the front end architecture via standard http methods.




Tech Stack

The database containing the news data was PSQL which I interacted with using Knex. For testing I used Jest and Supertest and the server was hosted on Heroku.

Features

  • Endpoints for each table in the database (topics, articles, users and comments)
  • Endpoints send responses in an object with a key name of what it is that is being sent as per the screenshot on the left hand side.
  • Some endpoints accept queries such as sorting and filtering features.
  • Endpoints are controlled using error handlers that produce the associated http status code and an appropriate error message
  • The server is built using strict MVC principles making it easy to identify problems and adding functionality over time
