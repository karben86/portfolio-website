Fully integrated HR system for Tiempo Espana Dance Academy


Description

As Tiempo Espana Dance Academy was a small company without a dedicated HR department, the aim was to create an application that dance teachers could use to effectively manage themselves automating as much as possible around administration, bookings, accounting etc. and only highlighting key issues to management.

Features

  • Internal job board advertising current positions for dance teachers who have access to the database
  • Password secured details page where dance teachers can manage their details such as address, phone number, bank details etc.
  • Management dashboard providing details on signup quotas, liability insurance and first aid certificates
  • Administration of teachers including set-up of new teachers and removal of past teachers
  • Ability for teachers to view any of their past contracts including earnings
  • Ability for teachers to update their details at any point
  • Teachers can create invoices directly from their account
  • Invoice dates default to prior month by default but can be customized to any time period
  • Invoices can be exported into a Word document or PDF
  • Calendar view where contracts are shown colour coded by availability
  • Teachers can sign up for any contract provided they have a valid insurance and first aid certificate and their level of experience matches the contract
  • Ability for teachers to find out further details about the contract i.e. location details, other teachers teaching at this venue etc.


Email Management Tool for Woodthorpe IT Solutions


Description

The aim of this project was to extract the text from emails as well as some of their meta data such as subject, date & time sent / received etc. into an Excel spreadsheet from any specified email address to allow users to easily archive email data as well as using stats and analysis tools from Excel that are not readily available in Outlook.

Features

  • Option to select which email address should be used to extract all emails
  • Basic filtering options to select how many emails should be extracted. By default all emails from all folders are extracted
  • Basic sorting options for some of the available fields in ascending or descending order
  • Results are printed out in a standard Excel spreadsheet where they can be analysed and exported further
  • Results contain fields such as To, CC, Subject, Body etc. as well as a field called "Type" which denotes the folder from which the email was extracted.
